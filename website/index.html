Features

Everything you need, in one place.

Inspired by Wavebox, Sidekick, and Zen Browser — open source, extensible, AI-native.

📑

Vertical Tabs

Tree-style vertical tabs with drag-and-drop grouping, search, pinning, suspend (discard memory), and close-history (restore last 50 closed tabs).

🤖

AI Agents

Built-in multi-agent system. Register agents with custom models/providers, assign tabs, automate browsing, track tasks, and view success rates.

⌨️

Command Palette

Ctrl+K fuzzy search across everything — workspaces, tabs, commands, settings, apps, and tasks. No mouse needed.

🪟

Split View

Side-by-side panels. Compare docs, monitor dashboards, or keep your AI visible while browsing. Multiple layout modes supported.

📋

Kanban Boards

Full kanban with boards, columns, cards, WIP limits, drag-and-drop, and color-coded statuses. Integrated into the browser sidebar.

🎯

Focus Mode

Pomodoro timer, distraction blocking, notification muting. Presets for 25/50/90 min sessions with short/long breaks.

📦

Workspaces

Isolated session partitions per workspace. Each workspace has its own tabs, apps, cookies, and kanban boards. Switch instantly.

🧩

App Shortcuts

Pin web apps to the sidebar with isolated accounts per app. Perfect for Slack, Gmail, Notion, GitHub, or any web app.

🔄

Auto-Update

Automatic updates via GitHub Releases (electron-updater). Always on the latest version with newest features.

🖥️

System Tray

Minimize to tray, quick actions (Show, Restart, Quit), server status indicator, and background operation.

🔒

Privacy Control

Built-in adblock, cookie controls (allow all / block third-party), per-app cache clearing, and telemetry toggle.

⚡

Resource Saver

Auto-suspend inactive tabs after N minutes. Configurable exceptions. Estimates ~50MB saved per suspended tab.

🎨

Theme System

Dark theme by default with customizable accent colors. Compact mode hides sidebar for more content area.

🛠️

Developer Tools

Built-in DevTools, JS console per tab, page content extraction, screenshot capability, and full IPC for automation.

🧰

Pipeline & Cron

Built-in cron scheduler and pipeline runner for automation workflows. Schedule tasks, run agents, trigger actions.
