Features

Everything you need.

Inspired by Wavebox, Sidekick, and Zen Browser — open source, extensible, AI-native.

📑

Vertical Tabs

Tree-style tabs with drag-and-drop grouping, search, pinning, tab suspension, and close-history (restore last 50 closed tabs).

🤖

AI Agents

Multi-agent system with any model/provider. Agents navigate tabs, click, type, scroll, screenshot, and extract content. Custom roles & prompts.

⌨️

Command Palette

Ctrl+K fuzzy search across workspaces, tabs, commands, settings, apps, tasks. No mouse needed.

🪟

Split View

Side-by-side panels. Compare docs, monitor dashboards, or keep your AI visible while browsing.

📋

Kanban Boards

Full kanban with boards, columns, cards, WIP limits, drag-and-drop, and color-coded statuses.

🎯

Focus Mode

Pomodoro timer (25/50/90 min) with distraction blocking, notification muting, and timer overlay.

📦

Workspaces

Isolated session partitions — each workspace has its own tabs, apps, cookies, and boards. Switch instantly.

🧩

App Shortcuts

Pin web apps with isolated accounts per app. Perfect for Slack, Gmail, Notion, GitHub.

🔄

Auto-Update

Automatic updates via GitHub Releases. Always on the latest version.
