System Requirements

  • OS: Windows 10 or later (x64)
  • RAM: 4 GB minimum (8 GB+ recommended)
  • Storage: ~500 MB for the app + ~1 GB for the Python backend and dependencies
  • Python: 3.10 or later (bundled in the installer, or use an existing installation)
  • Git: Optional, needed only for building from source
MacOS and Linux support are on the roadmap. The Electron app is cross-platform by nature, but the focus is Windows for now.

Installation

Option 1: Download the Installer (Recommended)

  1. Go to the GitHub Releases page and download the latest LastBrowser-Setup-x.y.z.exe.
  2. Run the installer. You'll be guided through the setup — choose your installation directory (or keep the default).
  3. Once installed, a desktop shortcut and Start Menu entry will be created automatically.
  4. Launch LastBrowser from the shortcut.

Option 2: Portable Version

Download LastBrowser-Portable-x.y.z.exe from the releases page. No installation needed — just run the executable from any folder (USB drive, cloud sync folder, etc.).

Option 3: Build from Source

See the Building from Source section below for full instructions.

First Run

When you first launch LastBrowser, you'll see:

  1. Splash Screen — "Starting LastBrowser" with loading modules (workspace, agents, sessions, services).
  2. Main Window — The dark-themed interface with the sidebar on the left and the main content area.
  3. Server Status — The Python WebUI backend starts automatically in the background. The status indicator in the sidebar footer shows "connected" when ready.

The default workspace is created automatically. From here you can:

  • Create new workspaces
  • Configure AI agents
  • Pin web apps to the sidebar
  • Start browsing with vertical tabs
  • Open the command palette (Ctrl+K)

Setting Up Workspaces

Workspaces keep your browsing organized. Each workspace has its own tabs, pinned apps, cookies, and kanban boards — fully isolated via Electron session partitions.

Create a Workspace

  1. Click the + button in the workspace switcher (top of the sidebar).
  2. Give it a name, pick an emoji icon, and choose an accent color.
  3. The new workspace appears in the switcher. Click to switch.

Manage Workspaces

  • Rename: Right-click a workspace name and select "Rename".
  • Reorder: Drag and drop workspaces in the switcher.
  • Pause: Temporarily pause a workspace to free resources.
  • Export/Import: Export a workspace configuration (tabs, apps, settings) and import it elsewhere.

Using the Sidebar

The sidebar is your navigation hub. All panels accessible from one place:

  • Chat — Interact with your AI agent
  • Kanban — Task management boards
  • Agents — AI agent configuration and monitoring
  • Browser — Browse with vertical tabs
  • Memory — View agent memory and recall
  • Skills — Manage agent skills
  • Settings — App-wide configuration

Use Ctrl+B to toggle sidebar visibility. The status footer shows server connection status, and quick-actions give you open-in-browser, restart, and quit.

Configuring AI Agents

LastBrowser includes a built-in multi-agent system. Each agent can be configured independently:

  1. Navigate to the Agents panel in the sidebar.
  2. Click Add Agent to create a new agent.
  3. Configure:
    • Name — A friendly identifier
    • Model — e.g., deepseek-v4-flash, gpt-4o, claude-sonnet-4
    • Provider — e.g., opencode-go, openai, anthropic
    • Max Tokens — Response length limit
    • Temperature — Creativity control (0.0 = deterministic, 1.0 = creative)
    • Role — Worker, Orchestrator, Supervisor, etc.
    • System Prompt — Custom instructions for the agent
  4. Enable the agent and assign it to tabs or tasks.

The agent dashboard shows real-time stats: total tasks run, tokens used, and success rate.

Using Kanban Boards

LastBrowser includes a full kanban system for task management:

  1. Navigate to the Kanban panel.
  2. Click New Board to create a board for the current workspace.
  3. Default columns are created: Inbox, Todo, Planned, In Progress, Waiting, Review, Testing, Done, Archived.
  4. Drag cards between columns to update status.
  5. Set WIP limits on columns to prevent bottlenecks.

Each card supports: title, description, assignee, labels, due date, checklist, and comments. Boards are workspace-scoped.

Focus Mode

Built-in Pomodoro-based focus system with distraction blocking:

  • Pomodoro (25m): Classic productivity sprint
  • Focus 50 (50m): Deep work session
  • Focus 90 (90m): Extended focus block
  • Short Break (5m): Quick recharge
  • Long Break (15m): Extended rest

When focus mode is active, notifications are muted, badges are hidden, and a timer overlay shows remaining time. Press Ctrl+Shift+F to toggle focus mode.

Building from Source

Prerequisites

Before building, ensure you have installed:

  • Node.js v18 or later (with npm)
  • Python 3.10 or later
  • Git

Clone & Build

git clone https://github.com/Loggableim/lastbrowser.git
cd lastbrowser/native
npm install
npm run dev   # Start in development mode

Build the Installer

npm run build     # Build NSIS installer (./native/dist/)
npm run build:dir # Build portable directory (./native/dist/)
npm run dist      # Production build for GitHub release

Development Workflow

# From the root workspace:
npm run desktop:dev    # Start Electron + Python backend in dev mode
npm run desktop:build  # Build the Windows installer
npm run icons          # Generate app icons (requires Python + Pillow)
The Python server (server.py) runs in the background. In dev mode, it auto-starts with the Electron app. In production, it's bundled as an extra resource.

FAQ

Q: Is LastBrowser free?
A: Yes. It's completely free and open source under the MIT license.

Q: Can I use it without an AI agent?
A: Yes. All browsing features work independently of the AI agent system.

Q: Does it work on Mac or Linux?
A: The core is cross-platform Electron, but Windows is the primary target for now. Mac/Linux support is planned.

Q: How do I update?
A: Updates are automatic via GitHub Releases. The app checks for updates on startup.
