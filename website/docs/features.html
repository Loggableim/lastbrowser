Workspaces Core

Workspaces are the foundation of LastBrowser's organization system. Each workspace has completely isolated state — its own tabs, pinned apps, cookies, kanban boards, and settings.

  • Session Isolation: Each workspace gets a unique Electron session partition (persist:workspace_<id>). Cookies, localStorage, and cache are fully separated.
  • Instant Switching: Switch between workspaces without losing state. Inactive workspaces are paused to save memory.
  • Export/Import: Export your entire workspace configuration (tabs, apps, board state) as JSON. Import on another machine.
  • Customization: Each workspace can have its own name, emoji icon, accent color, and tab grouping.

Use cases: Separate work and personal browsing. Create a workspace per project/client. Keep different AI agent configurations per workspace.

Vertical Tabs Core

LastBrowser uses vertical tabs (tree-style) by default — maximising horizontal space for content while keeping tabs readable and organized.

  • Tab Management: Create, close, duplicate, move between workspaces, restore from history.
  • Pin Tabs: Pin frequently used tabs. Pinned tabs are persistent and take minimal space.
  • Tab Suspension: Inactive tabs can be suspended (memory discarded) after a configurable timeout. ~50MB saved per suspended tab.
  • Close History: Last 50 closed tabs are saved per workspace. Ctrl+Shift+T to restore.
  • Navigation: Back, forward, reload, URL navigation, favicon tracking, and title updates.
  • Mute/Unmute: Per-tab audio control.
  • Zoom: Per-tab zoom level (persistent across sessions).

Sidebar & Navigation Core

The sidebar provides one-click access to all major features. It's always visible (unless hidden via Ctrl+B).

  • Panel Navigation: Chat, Kanban, Agents, Browser, Memory, Skills, Settings — each with its own icon and shortcut (Ctrl+1-8).
  • Workspace Switcher: Quick switch between workspaces with icons and status indicators.
  • Status Footer: Live server connection status (online/offline/restarting) with dot indicator.
  • Quick Actions: Open in external browser, restart server, quit app.
  • Pinned Apps: Your pinned web apps are accessible from the sidebar (see App Shortcuts).

AI Agent System Productivity

LastBrowser includes a sophisticated multi-agent system integrated directly into the browser interface. Each agent can be configured independently and assigned to tabs or tasks.

  • Multiple Agents: Create any number of agents with different configurations.
  • Provider-Agnostic: Supports any model provider — OpenAI, Anthropic, DeepSeek, OpenRouter, Google, or custom endpoints.
  • Role System: Each agent has a role (Worker, Orchestrator, Supervisor, Researcher, Coder, Writer) that defines its behavior.
  • Agent Dashboard: Real-time stats — total tasks, tokens used, success rate, enabled/disabled status.
  • Tab Integration: Assign agents to tabs. Agents can navigate, click, type, scroll, take screenshots, and extract content from assigned tabs.
  • Concurrent Tasks: Configure max concurrent tasks per agent. Workers run independently.
  • System Prompts: Custom system prompts for each agent to control behavior and output format.
// Example agent configuration
{
  name: "Code Reviewer",
  model: "deepseek-v4-flash",
  provider: "opencode-go",
  role: "reviewer",
  maxTokens: 8000,
  temperature: 0.3,
  enabled: true,
  maxConcurrentTasks: 2
}

Kanban Boards Productivity

A full kanban project management system built into the browser. No separate app needed.

  • Boards: Create multiple boards per workspace. Each board has its own columns and cards.
  • Columns: Default column set: Inbox → Todo → Planned → In Progress → Waiting → Review → Testing → Done → Archived. Each with a distinct color.
  • Cards: Each card has a title, description, assignee, labels, due date, checklist items, and comments.
  • WIP Limits: Set Work-In-Progress limits per column to prevent bottlenecks.
  • Drag & Drop: Cards can be reordered within columns and moved between columns.
  • Workspace-Scoped: Boards are isolated per workspace, matching the workspace's data partition.

Column colors by status: Inbox (#6b7280), Todo (#3b82f6), Planned (#8b5cf6), In Progress (#f59e0b), Waiting (#f97316), Review (#ec4899), Testing (#14b8a6), Done (#22c55e), Archived (#6b7280).

Split View Productivity

View two tabs side-by-side in a split layout. Perfect for comparison, research, or keeping your AI agent visible while working.

  • Layout Modes: 50/50 split (default), customizable panel sizes.
  • Panel Management: Add panels to the split, remove panels, reorder positions.
  • Active Tracking: Each panel tracks its active state, tab, and size independently.
  • Workspace-Scoped: Split view configuration is stored per workspace.

Focus Mode Productivity

Built-in Pomodoro-based focus system with automatic distraction blocking:

  • Timer Presets: Pomodoro (25m), Focus 50 (50m), Focus 90 (90m), Short Break (5m), Long Break (15m). Custom durations also supported.
  • Distraction Blocking: When active: notifications muted, badges hidden, sounds muted.
  • Timer State: Running/paused/stopped states with elapsed time tracking. Active session ID for IPC communication.
  • Settings Integration: Focus mode syncs with the global settings manager. When you exit focus, all distractions are re-enabled.

Command Palette Core

Press Ctrl+K to open the command palette — a fuzzy search across everything in LastBrowser.

  • Search Scope: Workspaces, tabs (by title/URL), commands, settings, apps, tasks, and sessions.
  • Fuzzy Matching: Smart scoring — exact matches score highest, followed by prefix, substring, and character-by-character fuzzy matching.
  • Quick Actions: Toggle focus mode, compact mode, mute all, toggle sidebar, open DevTools — all from the palette.
  • Navigation: Switch workspaces, activate tabs, open apps, run commands — all without touching the mouse.

App Shortcuts Productivity

Pin your most-used web apps to the sidebar for quick access:

  • Pinned Apps: Create shortcuts for Slack, Gmail, Notion, GitHub, Jira, or any web app.
  • Multiple Accounts: Add multiple accounts per app definition, each with its own isolated session (cookies, login state).
  • Categories: Organize apps by workspace.
  • Mute & Suspend: Mute audio per app, suspend/resume to save resources.
  • Badge Support: App badges show notification counts (coming soon).
  • Cache Control: Clear cache per app or globally.

Privacy & Security Core

  • Ad Blocking: Built-in adblock with configurable filter lists. Toggle on/off from settings.
  • Cookie Control: Allow all, block third-party cookies, or block all.
  • Cache Clearing: Clear cache and storage data per app or globally. Supports all session partitions.
  • Telemetry: Optional telemetry (off by default). Toggle in settings.
  • Session Isolation: Workspace and app session partitions prevent cross-site tracking.

Resource Saver Core

Automatically manages memory by suspending inactive tabs:

  • Auto-Suspend: Suspend tabs after N minutes of inactivity (default: 30).
  • Exceptions: Add URL patterns to the exception list to keep specific tabs always active.
  • Memory Stats: Dashboard shows total tabs, active/suspended count, and estimated memory saved (~50MB per suspended tab).
  • Manual Control: Sleep/wake individual apps, or suspend all inactive tabs at once.

Theme & Appearance Core

  • Dark Theme: Deep dark background (#0A0A12, #0D0D1A, #1A1A2E) with purple (#7C6FF0) and blue (#60A5FA) accents.
  • Compact Mode: Minimize chrome for more content area. Hides sidebar elements.
  • Customizable: Workspace-level accent color. Per-workspace icon.
  • Typography: Inter for UI, JetBrains Mono for code. Carefully tuned font sizes and spacing.

System Tray Core

  • Minimize to Tray: Close button minimizes to tray instead of quitting (configurable).
  • Quick Menu: Right-click tray icon for quick actions: Show Window, Restart Server, Quit.
  • Status: Server status indicator in the tray context menu.
  • Background Operation: The app and its Python backend continue running in the background.

Auto-Update Core

  • GitHub Releases: Updates are distributed via GitHub Releases using electron-updater.
  • Automatic: Checks for updates on startup. Downloads and installs in the background.
  • Release Channel: Configured for the Loggableim/lastbrowser repository.
  • Manual Check: Check for updates via Settings panel or tray menu.
