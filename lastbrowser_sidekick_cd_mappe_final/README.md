# lastbrowser + sidekick - Corporate Design Mappe v1.0

Enthalten:

- `brand-guide/pdf/BRAND_GUIDE_lastbrowser_sidekick_v1.pdf`
- `brand-guide/pages/` - alle PDF-Seiten als PNG
- `assets/logos/` - Logo-Dateien
- `assets/app-icons/` - App-Icon, PNG-Größen und ICO
- `assets/sidekick/` - Sidekick Mascot in Größen
- `assets/sidebar-icons/` - 30 Icons in png-original, png-512, png-256, png-128, png-64, png-32
- `tokens/` - Farben als CSS/JSON/Tailwind JSON
- `index.html` - schnelle Asset-Galerie
- `preview/` - Cover und Icon Contact Sheet

Empfohlene Nutzung:

- Sidebar: png-32 oder png-64
- Cards/Launcher: png-128 oder png-256
- Marketing/Docs: png-512
- Windows Shortcut/Installer: assets/app-icons/Lastbrowser.ico
