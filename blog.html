Website v3 May 13, 2026

New Website: PWA, Dark Theme, Docs Search

The lastbrowser.pawsunited.info website got a massive overhaul. New features include: PWA support (installable app), mobile hamburger menu, scroll-to-top button, Intersection Observer scroll animations, Fuse.js-powered documentation search, self-hosted fonts (no Google dependency), CSS extraction, privacy policy and imprint, German translation, and a live GitHub stats banner. All 14 pages now load with external CSS and favicon.

Release May 13, 2026

LastBrowser v0.2.0 Released

Major update to the v2 architecture. Complete module system with 21 managers, integrated sidebar workspace HTML, multi-agent system with configurable model providers, kanban boards with WIP limits, split view with multi-panel layouts, Pomodoro focus mode, Ctrl+K command palette with fuzzy search, workspace session isolation, system tray, and auto-update via GitHub Releases. See the full changelog for details.

Project April 2026

LastBrowser: A New Open-Source Browser

LastBrowser started as the Hermes Desktop native wrapper and evolved into a full productivity workspace browser. Inspired by Wavebox, Sidekick, and Zen Browser, the goal is to build the last browser you'll ever need — open source, AI-native, and built for deep work. The roadmap shows where we're headed: cross-platform support, AI agent marketplace, and v1.0 production readiness.
